# Proving a Unicorn is magical

1. If the unicorn is mythical then it is immortal
2. If the unicorn is not mythical then it is a mortal mammal
3. If the unicorn is either immortal or a mammal then it is horned
3. The unicorn is magical if it is horned

---
Propositional logic:

isMythical => isImmortal
¬isMythical => isMammal
isImmortal v isMammal => isHorned
isMagical <=> isHorned
