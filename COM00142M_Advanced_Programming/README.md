# COM00142M Advanced Programming

* Week 1: Introduction to Python
* Week 2: Data sources
* Week 3: Event driven programming and GUI
* Week 4: APIs for Data Preparation
* Week 5: Professionalism
* Week 6: Machine Learning with Neural Network
* Week 7: APIs for Data Wrangling and Visualisation

At the end of this module students will be able to:
* Demonstrate a critical understanding of the theory and application of advanced programming techniques
* Design and implement programs for real-world problems
* Communicate design decisions for the selection, storage and manipulation of data
* Critically evaluate the legal and ethical impact of software developments in real-world contexts.
